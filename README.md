## Описание
**GitHub User Finder** - это десктопное приложение с графическим интерфейсом для поиска пользователей GitHub и управления списком избранных. 

Приложение использует **только встроенные библиотеки Python**, не требуя установки дополнительных зависимостей. Оно демонстрирует работу с:
- GitHub API через HTTPS запросы
- JSON для хранения данных
- Tkinter для графического интерфейса
- Файловой системой

## Возможности

### Основной функционал
- 🔍 **Поиск пользователей** GitHub по имени или логину
- 📋 **Просмотр информации** о пользователе:
  - Имя пользователя (логин)
  - Полное имя
  - Количество публичных репозиториев
  - Количество подписчиков
- ⭐ **Добавление в избранное** понравившихся пользователей
- ❌ **Удаление из избранного**
- 💾 **Автоматическое сохранение** избранных пользователей в JSON файл
- ✅ **Валидация ввода** - поле поиска не может быть пустым

### Технические особенности
- Использует только встроенные модули Python
- Не требует установки дополнительных пакетов
- Работает через HTTPS с GitHub API
- Поддерживает обработку сетевых ошибок
- Сохраняет дату добавления в избранное
